### 外部のイベントをフックにしてワークフローを実行する

1. GitHubのアクセストークンを発行する
2. 外部サービスにトークンを登録する
3. 外部サービスにトリガーイベント名を登録する
4. GitHub Actionsのworkflowに以下のように`repository_dispatch`を設定する

```yml
on:
  repository_dispatch:
    types: [トリガーイベント名]
```
