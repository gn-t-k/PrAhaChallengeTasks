### ログレベルとは

サーバーに起きた事象を記録することを「ログ」と呼び、内容の重要度によってレベル分けすることをログレベルという。

例として、PSR-3（PHPの規約）では以下のように定義されている。

| ログレベル | 説明 | 例 |
| - | - | - |
| emergency | システムが使用不可能な状態 | - |
| alert | 直ちに何らかの対処の必要がある |  |
| critical | 危機的な状態 |  |
| error | 直ちに対処する必要のない実行時エラーだが、通常はログに記録して監視すべき |  |
| warning | エラーではない例外的な出来事 |  |
| notice | 正常だが、重要な事象 |  |
| info | 興味深い事象 |  |
| debug | 詳細なデバッグ情報 |  |
